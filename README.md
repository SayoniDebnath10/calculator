# Calculator (Vanilla JavaScript)
*Current Version: *v1.0

A simple calculator built using HTML, CSS and JavaScript.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Decimal operations
- Performs operations on two numbers at a time
- Clear screen

## Upcoming Features
- Responsive Design
- Continuous Expressions
- Calculation History
  
## Tech Stack
- HTML5
- CSS3
- JavaScript
